Machine Learning Survival Models

Random Survival Forest, Gradient Boosted Survival, SHAP interpretability

Run setup Run one RSF, GBS, or LASSO-Cox model here. Compare All is a separate screening run.
Evaluation These settings apply to Compare All screening only. Run Analysis uses its own holdout/apparent path.
Compare All is a screening comparison across Cox PH and, when available, LASSO-Cox, RSF, and GBS on one shared evaluation path. Run Analysis fits one RSF, GBS, or LASSO-Cox model and reports its own holdout/apparent evaluation mode in the result banner. Fast mode skips SHAP so RSF and GBS return faster on larger cohorts. If TreeExplainer is unavailable, SHAP falls back to an approximate screening view rather than a manuscript-ready attribution. SHAP safe mode can refit a reduced companion tree model when the encoded feature matrix is too wide, and that companion run is labeled explicitly.
As a rough interpretation guide, a C-index near 0.50 is chance-level ranking. Values above about 0.70 can be useful for screening, but external validation matters more than thresholds.
Partial dependence and counterfactual analysis are available for tree-model runs only (RSF or GBS). They reuse the latest matching trained tree model when available; otherwise they refit the same requested tree settings.
